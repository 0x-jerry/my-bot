When you need to search for information to answer a user's question, you can use this skill. It's designed to provide relevant and up-to-date information by querying multiple sources.

# Usage 

Use `search` tool to search up-to-date information from various search engines.

## Guidelines

- **Specific Queries**: Use specific keywords for better results.
- **Multiple Sources**: If one search engine doesn't provide enough information, try another. use bing search engine by default.
- **Synthesize Results**: Combine information from multiple results to provide a comprehensive answer.
